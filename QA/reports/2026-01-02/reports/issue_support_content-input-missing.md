## 문제 설명
고객센터 페이지(`/support`)의 "문의하기" 섹션에서 "내용" 입력 필드가 없습니다. "내용" 섹션에는 안내 텍스트만 표시되고 실제로 내용을 입력할 수 있는 textarea나 input 필드가 존재하지 않습니다.

## 재현 단계
1. 로그인 후 "고객센터" 메뉴 클릭
2. "문의하기" 섹션 확인
3. "내용" 섹션에 입력 필드가 없는지 확인

## 예상 동작
"내용" 섹션에 textarea 또는 contenteditable 영역이 있어 사용자가 문의 내용을 입력할 수 있어야 합니다.

## 실제 동작
"내용" 섹션에는 안내 텍스트만 표시되고, 실제 입력 필드가 없어 내용을 입력할 수 없습니다.
